# Phone Lookup Tool

A simple web application to lookup information about phone numbers using the Truecaller API.

## Features

- Clean and modern user interface
- Phone number validation (10-digit Indian numbers)
- Real-time phone number lookup
- Displays caller information including name, carrier, and location
- Responsive design

## Usage

1. Open `index.html` in your web browser
2. Enter a 10-digit phone number
3. Click "Lookup Number" to fetch information
4. View the results or perform a new search

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling and layout
- `script.js` - Phone lookup logic and API integration

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- UnOfficial Truecaller API (via proxy)
